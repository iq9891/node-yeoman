# yeoman

This project is generated with [yo angular generator](https://github.com/yeoman/generator-angular)
version 0.11.1.

## 文件配置 ##
1. 安装node模块
<pre>npm install</pre>
2. 安装引用js
<pre>bower install</pre>

## Build & development ##

Run `grunt` for building and `grunt serve` for preview.

## 测试 ##

运行 `grunt test` 将运行单元测试与因果报应.


## 预览 ##
运行 grunt serve 可以预览
